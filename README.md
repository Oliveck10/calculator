# calculator-prototype
- Using webpack, babel as env config
- Using react as front-end framework

## install
```
npm install
```

## build
```
npm run build
```

## start
```
npm start
```
Then go to `http://localhost:3000`

## lint
```
npm run lint
```

## test
```
npm run test
```
